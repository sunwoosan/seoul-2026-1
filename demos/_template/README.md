# 학급 평가 결과 분석 도구

학급 평가 결과 CSV를 업로드하면 표와 그래프로 데이터를 빠르게 파악할 수 있는 Streamlit 앱입니다.
수행 평가, 단원 평가 등의 결과를 시각화하여 학생 수준을 신속하게 확인하고, 수준별 학습지 제공을 위한 참고 자료로 활용할 수 있습니다.

> 2026-1학기 클라우드 프로그래밍

---

## 1. 무엇을 할 수 있나요

1. **CSV 업로드 + 데이터 확인**
   - 사이드바에서 학급 평가 결과 CSV 파일을 업로드할 수 있습니다.
   - 표 위에 `st.metric`으로 **응답자 수**가 크게 표시됩니다.
   - 업로드한 CSV 전체 내용을 표로 확인할 수 있습니다.

2. **문항별 응답 분포 확인**
   - 드롭다운에서 문항 이름을 선택할 수 있습니다.
   - 선택한 문항의 응답 분포가 **Plotly 막대그래프**로 표시됩니다.
   - 각 응답별 개수도 함께 표로 확인할 수 있습니다.

3. **전체 요약 보기**
   - `전체 요약 보기` 버튼을 누르면 모든 숫자형 문항이 점수대별로 재배치됩니다.
   - 점수대별 결과를 표와 그래프로 함께 확인할 수 있습니다.

## 2. 입력 CSV 형식

아래와 같이 문항별 응답 또는 점수가 들어 있는 CSV 파일을 사용합니다.

| 컬럼명 | 타입 | 예시 | 필수 |
|--------|------|------|------|
| `이름` | 문자열 | `학생1` | 선택 |
| `평가명` | 문자열 | `단원평가 1` | 선택 |
| `문항1` | 문자열/숫자 | `O`, `X`, `5` | 권장 |
| `문항2` | 문자열/숫자 | `상`, `중`, `하`, `80` | 권장 |
| `점수` | 숫자 | `85` | 권장 |

- 기능 2는 모든 컬럼을 문항 후보로 사용할 수 있습니다.
- 기능 3은 **숫자형 컬럼만** 점수대별 요약 대상으로 사용합니다.
- 샘플 파일: [`sample_data.csv`](sample_data.csv)

## 3. 사용 방법

1. 왼쪽 사이드바에서 CSV 파일을 업로드합니다.
2. **① 데이터 확인**에서 응답자 수와 표를 확인합니다.
3. **② 문항별 응답 분포**에서 문항을 선택하고 Plotly 그래프를 확인합니다.
4. **③ 전체 요약 보기**에서 버튼을 눌러 숫자형 문항의 점수대별 분포를 확인합니다.

## 4. 로컬 실행

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## 5. Streamlit Community Cloud 배포

1. GitHub public 저장소에 코드를 푸시합니다.
2. Streamlit Community Cloud에서 **New app**을 선택합니다.
3. 저장소와 `demos/_template/app.py` 파일을 선택합니다.
4. Deploy를 누릅니다.
5. 배포가 완료되면 URL이 생성됩니다.

## 6. 완료 조건 (Definition of Done)

아래 4가지가 모두 만족되면 완료입니다.

- 로컬(내 노트북)에서 `streamlit run app.py`로 에러 없이 실행된다.
- Streamlit Community Cloud에 배포되어 URL이 생성되었다.
- 샘플 데이터로 위 3개 기능이 모두 동작한다.
- `README.md`에 사용법이 한글로 설명되어 있다.

## 7. 개인정보 / 보안

- 샘플 데이터에는 실제 학생 이름 / 개인정보를 넣지 않습니다.
- API 키 / 비밀번호를 `app.py`에 직접 작성하지 않습니다.
- Public 저장소에 올려도 무방한 데이터만 사용합니다.
- `.gitignore`에 민감 파일이 포함되어 있는지 확인합니다.
- 실제 학생 이름이 포함된 CSV는 저장소에 업로드하지 않는 것을 권장합니다.
