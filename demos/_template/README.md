# 학급 평가 결과 분석 도구

학급 평가 결과 CSV를 업로드하면 표와 그래프로 데이터를 빠르게 파악할 수 있는 Streamlit 앱입니다.  
수행 평가, 단원 평가 등의 결과를 시각화하여 학생 수준을 신속하게 확인하고, 수준별 학습지 제공을 위한 참고 자료로 활용할 수 있습니다.

> 2026-1학기 클라우드 프로그래밍

---

## 1. 무엇을 할 수 있나요

이 앱은 학급 평가 데이터를 업로드한 뒤 다음 기능을 제공합니다.

### ① 데이터 확인
- CSV 파일을 업로드할 수 있습니다.
- 업로드한 데이터의 **응답자 수**를 큰 숫자로 확인할 수 있습니다.
- 전체 데이터를 표로 바로 확인할 수 있습니다.

### ② 점수 분포 확인
- **숫자형 컬럼만 선택**해서 점수 분포를 볼 수 있습니다.
- 선택한 점수 컬럼을 기준으로 점수대를 아래와 같이 나누어 막대그래프로 보여줍니다.
  - 0~59
  - 60~69
  - 70~79
  - 80~89
  - 90~100
- 각 점수대별 학생 수를 표로도 확인할 수 있습니다.

### ③ 전체 요약 보기
- 모든 숫자형 컬럼을 한 번에 요약해서 볼 수 있습니다.
- 각 문항(점수 컬럼)에 대해 아래 정보를 확인할 수 있습니다.
  - 응답 수
  - 평균
  - 최고점
  - 최저점
- 전체 점수를 점수대별로 재배치한 표와 그래프도 함께 확인할 수 있습니다.

### ④ 학생 수준 분류표
- `번호` 컬럼과 선택한 점수 컬럼을 기준으로 학생 수준을 자동 분류합니다.
- 수준은 다음 4단계입니다.
  - 노력요함
  - 보통
  - 잘함
  - 매우잘함
- 수준별 학생 수 그래프도 함께 보여줍니다.

### ⑤ 수준별 피드백 제공
- 학생 번호를 선택하면 해당 학생의:
  - 점수
  - 수준
  - 피드백
  을 확인할 수 있습니다.
- 수준별 피드백은 다음과 같습니다.
  - **노력요함**: 교과서, 배움공책 내용을 좀 더 꼼꼼하게 복습하세요.
  - **보통**: 보통맛 학습지를 한 번 더 풀어보세요.
  - **잘함**: 매운맛 학습지를 한 번 더 풀어보세요.
  - **매우잘함**: 핵불닭맛 학습지를 풀어보세요.

### ⑥ 기준 점수 미달 학생 목록
- 선택한 점수 컬럼에서 **70점 미만 학생 목록**을 추출할 수 있습니다.
- 미달 학생 수와 학생 목록을 표로 확인할 수 있습니다.

## 2. 입력 CSV 형식

이 앱은 평가 결과가 담긴 CSV 파일을 입력으로 사용합니다.

### 기본 예시

| 컬럼명 | 타입 | 예시 | 설명 |
|---|---|---|---|
| `번호` | 숫자/문자열 | `1`, `2`, `3` | 학생 식별용 컬럼 |
| `평가명` | 문자열 | `단원평가1` | 평가 이름 |
| `점수` | 숫자 | `85` | 대표 점수 컬럼 |
| `국어` | 숫자 | `77` | 과목/문항 점수 |
| `수학` | 숫자 | `92` | 과목/문항 점수 |

### 예시 CSV

```csv
번호,평가명,국어,수학,점수
1,단원평가1,78,92,85
2,단원평가1,64,71,68
3,단원평가1,55,60,58
```

### 입력 시 주의할 점
- **학생별 기능(수준 분류, 피드백, 미달 학생 목록)** 을 사용하려면 `번호` 컬럼이 필요합니다.
- 점수 분포, 전체 요약 기능은 **숫자형 컬럼**이 있어야 동작합니다.
- 문자열만 있는 CSV를 업로드하면 점수 관련 기능은 동작하지 않습니다.
- 한글 CSV 인코딩은 `utf-8-sig`, `utf-8`, `cp949`, `euc-kr`를 자동으로 시도합니다.

## 3. 사용 방법

### 1단계. CSV 업로드
왼쪽 사이드바에서 학급 평가 결과 CSV 파일을 업로드합니다.

### 2단계. 데이터 확인
`① 데이터 확인` 영역에서 응답자 수와 업로드된 전체 데이터를 확인합니다.

### 3단계. 점수 분포 확인
`② 점수 분포`에서 보고 싶은 점수 컬럼을 선택합니다.  
선택한 컬럼의 점수대별 학생 수를 그래프와 표로 확인할 수 있습니다.

### 4단계. 전체 요약 보기
`③ 전체 요약 보기` 버튼을 누르면 모든 숫자형 컬럼에 대한 기본 통계와 점수대 재배치 결과를 확인할 수 있습니다.

### 5단계. 학생 수준 분류 확인
`④ 학생 수준 분류표`에서 점수 컬럼을 선택하면 학생별 수준과 피드백을 확인할 수 있습니다.

### 6단계. 개별 피드백 확인
학생 번호를 선택하면 해당 학생의 점수, 수준, 피드백이 펼침형으로 표시됩니다.

### 7단계. 미달 학생 목록 확인
`⑤ 기준 점수 미달 학생 목록`에서 점수 컬럼을 선택하면 **70점 미만 학생 목록**을 확인할 수 있습니다.

## 4. 로컬 실행 방법

아래 명령으로 로컬에서 실행할 수 있습니다.

```bash
cd demos/_template
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

### Windows PowerShell 사용 시

```powershell
cd demos/_template
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

## 5. Streamlit Community Cloud 배포 방법

1. GitHub 저장소에 코드를 푸시합니다.
2. Streamlit Community Cloud에 로그인합니다.
3. **New app**을 선택합니다.
4. 아래 값을 입력합니다.
   - **Repository**: `gonano9595/seoul-2026-2`
   - **Branch**: `main` 또는 배포하려는 브랜치
   - **Main file path**: `demos/_template/app.py`
5. Deploy를 누르면 배포가 시작됩니다.

### 필요한 패키지
배포를 위해 `demos/_template/requirements.txt`에 아래 패키지가 포함되어 있어야 합니다.

- streamlit
- pandas
- plotly
