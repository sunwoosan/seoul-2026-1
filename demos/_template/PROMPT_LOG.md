# PROMPT_LOG.md

GitHub Copilot Chat에 입력한 프롬프트와 반영 결과를 시간 순서대로 기록합니다.

---

## #1 — 점수 분포 중심으로 기능 수정

> **Prompt**
> 1. 문항별 응답 분포 대신 점수 분포로 바꾸고 싶습니다.
> 2. 숫자형 컬럼만 선택 가능하게 수정하고 싶습니다.
> 3. 전체 요약에 평균/최고점/최저점을 추가하고 싶습니다.

**결과:**
- 기존 문항별 응답 분포를 점수 분포 기능으로 변경했다.
- 점수 선택 드롭다운에서 숫자형 컬럼만 보이도록 수정했다.
- 전체 요약에 문항별 응답 수, 평균, 최고점, 최저점을 표시하도록 확장했다.

---

## #2 — 학생 수준 분류표와 미달 학생 목록 추가

> **Prompt**
> 1. 학생 수준 분류표 자동 생성을 하고 싶습니다.
>    (노력요함, 보통, 잘함, 매우잘함 네 항목으로)
> 2. 기준 점수 미달 학생 목록을 추출하고 싶습니다.
>    (60점 미만)

**결과:**
- 학생 수준 분류표 기능을 추가했다.
- 점수에 따라 노력요함 / 보통 / 잘함 / 매우잘함으로 자동 분류되도록 구현했다.
- 기준 점수 미달 학생 목록 기능을 추가했다.

---

## #3 — 기준 점수 변경

> **Prompt**
> 기준 점수 미달에서의 기준 점수를 70점 미만으로 수정하고 싶습니다.

**결과:**
- 미달 학생 목록의 기준 점수를 60점 미만에서 70점 미만으로 변경했다.

---

## #4 — 수준별 피드백 기능 추가

> **Prompt**
> 제가 전달하고 싶은 피드백은 다음과 같습니다.
>
> - 노력요함 : 교과서, 배움공책 내용을 좀 더 꼼꼼하게 복습하세요.
> - 보통 : 보통맛 학습지를 한 번 더 풀어보세요.
> - 잘함 : 매운맛 학습지를 한 번 더 풀어보세요.
> - 매우잘함 : 핵불닭맛 학습지를 풀어보세요.
>
> 교사나 학생이 점수를 클릭했을 때 피드백이 팝업처럼 뜨는 형식으로 전달하고 싶습니다.

**결과:**
- 수준별 피드백 문구를 추가했다.
- 학생 수준 분류표에 피드백 컬럼을 추가했다.
- Streamlit 환경에 맞게 학생 번호를 선택하면 expander에서 점수 / 수준 / 피드백을 확인할 수 있도록 구현했다.

---

## #5 — 개인정보 보호를 위한 학생 식별 컬럼 변경

> **Prompt**
> 개인 정보 보호를 위해 "이름" 컬럼을 "번호"로 수정하고 싶습니다.

**결과:**
- 학생 식별 컬럼을 `이름`에서 `번호`로 변경했다.
- 예시 CSV 컬럼 안내도 `번호` 기준으로 수정했다.
- 수준 분류표, 미달 학생 목록, 개별 피드백 조회 기능을 모두 `번호` 기준으로 변경했다.
